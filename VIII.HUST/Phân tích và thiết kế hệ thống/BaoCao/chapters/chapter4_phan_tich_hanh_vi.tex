\section{Giới thiệu giai đoạn}

\textbf{Phân tích cấu trúc} trả lời: hệ thống gồm những \textbf{lớp đối tượng} nào, thuộc tính và quan hệ ra sao. Biểu đồ đặc trưng: \textbf{Class Diagram} (biểu đồ lớp).

\begin{itemize}[leftmargin=1.5cm]
    \item Trước đó: Ch.3 \textbf{Phân tích chức năng} -- Use Case.
    \item Sau đó: Ch.5 \textbf{Phân tích hành vi} -- Sequence.
\end{itemize}

\section{Các lớp chính}

Từ nghiệp vụ và use case, xác định các lớp:

\begin{itemize}[leftmargin=1.5cm]
    \item \texttt{NguoiDung} -- tài khoản, phân quyền (UC01--UC03).
    \item \texttt{HangHoa}, \texttt{NhomHangHoa}, \texttt{DonViTinh} -- danh mục hàng (UC06--UC07).
    \item \texttt{NhaCungCap}, \texttt{KhachHang} -- đối tác (UC04--UC05).
    \item \texttt{Kho}, \texttt{TonKho} -- kho và tồn (UC08, nhập/xuất/kiểm kê).
    \item \texttt{PhieuNhapKho}, \texttt{ChiTietPhieuNhap} -- UC09.
    \item \texttt{PhieuXuatKho}, \texttt{ChiTietPhieuXuat} -- UC10.
    \item \texttt{PhieuKiemKe}, \texttt{ChiTietKiemKe} -- UC11.
    \item \texttt{LichSuThaoTac} -- ghi log.
\end{itemize}
